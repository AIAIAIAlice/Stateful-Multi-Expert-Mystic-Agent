# 八字排盘计算规则

本项目的八字排盘入口为 `metaphysics_agent.bazi_calculator.calculate_bazi`，MCP 工具入口为 `bazi_calculator`。

## 输入

- `birth_date`: 公历日期，格式 `YYYY-MM-DD`。
- `birth_time`: 24 小时时间，格式 `HH:MM`。
- `birth_place`: 出生地名称。
- `timezone`: IANA 时区或 UTC 偏移，例如 `Asia/Shanghai`、`+08:00`。
- `latitude` / `longitude`: 出生地坐标；启用真太阳时时必须提供经度，内置城市表可自动补全。
- `true_solar_time`: 是否启用真太阳时。默认 `false`。

## 四柱规则

- 年柱：以立春交节时刻作为干支年的分界。立春前仍属上一干支年，立春后进入新干支年。
- 月柱：以节气中的“节”作为月令边界，不按公历月份切换。寅月始于立春，卯月始于惊蛰，辰月始于清明，依次类推。
- 日柱：使用 `lunar-python` 的 `getDayInGanZhiExact`，采用晚子时 `23:00-23:59` 归次日的流派。
- 时柱：以两小时为一个时辰，时干由日干推导。

## 节气来源

节气时刻来自 `lunar-python>=1.4.8` 的 JieQi 表。项目代码只使用节气边界推导年柱、月柱，不使用 hash、随机数或伪造映射生成四柱。

## 真太阳时

默认按输入时区的标准时间排盘。启用 `true_solar_time` 时：

- 经度修正：`4 * (出生地经度 - 时区标准经线)` 分钟。
- 均时差：使用 NOAA 常用近似公式估算。
- 调整后的时间会用于年、月、日、时柱计算，并写入 `calculation_metadata.true_solar_time`。
